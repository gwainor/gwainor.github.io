About me

I'm Gökhan, a lead full-stack developer with over two decades in tech—ever since I built my first website in 2002 (it's still online! Check it out, source code). My fascination with computers began at age ten, and I've been immersed in coding ever since, learning web development through offline HTML pages, the PHP manual, and open-source projects.

Over the years, I've created intranet apps for large organizations like OYAK and Koç, dove into AI-powered startups in the U.S., and built solutions spanning web, mobile, and backend systems. I'm a relentless problem solver who loves turning ideas into robust, user-centered experiences.

I'm a passionate developer who loves to code and build things. I have worked with many programming languages and frameworks.
